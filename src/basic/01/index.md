# 1주차. 자연언어 처리 소개

1주차 과정에서는 자연언어처리의 거시적 개념과 패러다임 변화를 다룹니다.

* **[섹션 01. 자연언어처리와 텍스트 마이닝의 본질](/basic/01/sec01/)**
  * 분석가와 통역사의 차이
  * 자연언어처리의 궁극적인 목적
* **[섹션 02. 자연언어의 끔찍한 모호성과 텍스트 특성](/basic/01/sec02/)**
  * 컴퓨터가 멘붕에 빠지는 세계: 2가지 언어 대립
  * 확장되는 자연언어의 세계 (모든 기호 = 텍스트)
  * 자연어를 정복하기 위한 4대 기계 임무 (Tasks)
  * 사람을 미치게 하는 텍스트의 성질 (비정형 데이터)
  * 텍스트 마이닝의 원동력: 정보의 대폭발
* **[섹션 03. 텍스트 마이닝 공정과 고전 분석 방식의 한계](/basic/01/sec03/)**
  * 비정형 쓰레기를 황금으로: 통계적 기계 작동 4단계
  * 고대의 NLP: 어설펐던 룰 베이스(Rule-based)의 시대
  * 과거 고전적 분석 방식의 치명적 몰락 원인
* **[섹션 04. 트랜스포머의 제국 통일과 LLM의 환각](/basic/01/sec04/)**
  * 세 번째 패러다임: LLM 기반의 세계 정복 (현대 NLP 대통합)
  * 트랜스포머의 심장: 셀프 어텐션(Self-Attention)의 무서운 눈치술
  * 딥러닝계 최고의 사기꾼, '환각(Hallucination)' 현상
  * 환각을 묶는 인간의 목줄 방어막 도구 (RAG & RLHF)
  * 현업 엔지니어의 최후의 도구 선택 잣대
* **[섹션 05. 자연어 전처리와 벡터화 수치 계산 기초](/basic/01/sec05/)**
  * 기계의 뱃밥을 짓다: 데이터 전처리의 세부 공정 3단계
  * 가장 무식하고 원시적인 번역: 원-핫 인코딩 (One-hot encoding)
  * 문법 따위 무시하고 쪽수로 승부한다: BoW 패러다임
* **[섹션 06. 딥러닝(NLP)의 화려한 실무 산업 활용 사례](/basic/01/sec06/)**
  * 문서 요약 프레스 기계 (Summarization)
  * 오싹한 감성 분석기 (Sentiment Analysis)
  * 눈이 달린 초월적 언어모델 (Multi-modal: VQA, Text-to-Image)
