# 6주차. 텍스트 분류와 모델 성능 평가 지표

단어를 벡터 엑셀 표로 만드는 전단계를 넘어, 기계가 이 문서를 씹어 삼켜서 '긍정', '부정', '스팸' 방으로 던져 넣는 텍스트 분류 모델(나이브 베이즈, 로지스틱)과 그들의 정확도 사기극을 감별하는 교차 채점 지표 기술을 배웁니다.

* **[섹션 01. 기계학습: 텍스트 분류의 본질과 파이프라인](/basic/06/sec01/)**
  * 분류 모터(학습/예측)의 이중 심장 엔진 구조
  * 피처 추출기(BoW 상기)의 역할
* **[섹션 02. 멍청한 통계의 기적: 나이브 베이즈 분류기 (Naive Bayes)](/basic/06/sec02/)**
  * 앞뒤 맥락을 포기하는 나이브 엑셀 주사위
  * 분모가 $0$이 되는 폭발을 막는 스무딩 기법
* **[섹션 03. 결정 경계의 마법: 로지스틱 회귀와 소프트맥스 압축](/basic/06/sec03/)**
  * 이진 분류기의 꽃: 0%와 100%를 그리는 시그모이드 S-곡선
  * 다중 카테고리 100% 피자 분배를 위한 소프트맥스(Softmax)
* **[섹션 04. 모델의 오만함 판별: 모의고사와 K-Fold 교차 검증](/basic/06/sec04/)**
  * 과적합 부정행위를 막는 데이터 3분할 홀드아웃(Hold-out)
  * 극악의 난이도 불운을 타파하는 K-Fold 5조각 회전초밥 룰렛
* **[섹션 05. 오만한 정답률의 사기극: 혼동 행렬의 진실 방](/basic/06/sec05/)**
  * 랜섬웨어를 100% 놓지는 99.9% 깡통 스팸 분류기의 위협
  * 치명적 오작동 4구역: TP, TN, FP(억울한 시민), FN(방관 경찰)
* **[섹션 06. 진실을 꿰뚫는 지표의 파괴력: 정밀도, 재현율, F1, ROC/AUC](/basic/06/sec06/)**
  * 사기 수사 억울함 방지 (정밀도) vs 테러범 그물 투망 사냥 (재현율)
  * 극단적 치팅을 방어하는 조화 평균 F1 스코어의 파괴력
  * 임계값 조작 꼼수를 막는 ROC 곡선 그리기와 AUC 덩치 분석
