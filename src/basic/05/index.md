# 5주차. 워드 임베딩: 대수학 공간으로의 도약

통계적 빈도수를 세던 구시대 기법을 넘어서, 단어의 숨겨진 의미와 문맥을 파악하여 인공지능의 뇌(기하학적 벡터 공간)에 쏘아 올리는 황홀한 "워드 임베딩" 기술의 눈부신 진화(NNLM $\to$ Word2Vec $\to$ GloVe $\to$ FastText)를 배웁니다.

* **[섹션 01. 차원의 저주와 임베딩(Embedding)의 좌표 마법](/basic/05/sec01/)**
  * 희분한 원-핫 인코딩의 치명적 한계 (상호 직교)
  * 기하학적 유사도 추론 방정식 (King - Man = Queen)
* **[섹션 02. 통계의 종말: 투사층 룩업 테이블 트릭 (NNLM)](/basic/05/sec02/)**
  * 뉴럴 네트워크 언어모델(NNLM)의 첫 등장
  * GPU 연산을 아끼는 마법의 룩업 테이블(Lookup Table) 매핑
* **[섹션 03. 구글 워드 임베딩 제국 1부 (CBOW와 Skip-gram)](/basic/05/sec03/)**
  * 극단적 경량화로 은닉층을 도려낸 얕은 신경망 구조
  * 빈칸 힌트 탐색기 CBOW와 독고다이 추론기 Skip-gram의 위력
* **[섹션 04. 구글 워드 임베딩 제국 2부 (SGNS 네거티브 샘플링)](/basic/05/sec04/)**
  * 구글 서버를 마비시킬 뻔한 소프트맥스의 분모 폭발
  * 무작위 오답 갱신 트릭: 네거티브 샘플링(SGNS)의 승리
* **[섹션 05. 미지 단어(OOV)의 방어막: 서브워드 패키지 (FastText)](/basic/05/sec05/)**
  * 오타와 신조어 앞에 무너진 Word2Vec의 약점
  * 철자 조각(n-gram) 레고 블록 합산으로 OOV를 막아내는 페이스북의 역습
* **[섹션 06. 신경망과 통계의 이종교배: 스탠포드 GloVe (Global Vectors)](/basic/05/sec06/)**
  * 좁은 창문(Local Window) 편식에 대한 스탠포드의 일침
  * 전체 문헌의 동시 등장 엑셀 행렬과 오차 함수의 만남
