# 07. 텍스트 주제 분석과 차원 압축 조물주 (LSA / LDA)

꼬리표가 당당하게 붙어있던 온실 속 분류 화초의 세계를 벗어나, 인간의 정답지 도움 없이 세상의 수만 장 야생 뉴스들이 어떤 공통된 카테고리를 통계적으로 파벌을 이루고 있는지 기계 개척자가 홀로 눈치채게 만드는 **"비지도 학습(Unsupervised 군집화)"**의 황무지 정수를 들이마십니다.

행렬을 프레스로 박살 내어 차원의 저주를 증발시키는 잠재의미분석(LSA)의 거친 수학부터, 애초에 문서를 무작위 주사위 확률의 피조물로 취급하고 MCMC 깁스 커닝을 통해 조물주의 비율표를 훔쳐보는 최강의 베이즈 통계학 잠재 디리클레 할당(LDA) 확률 역추론 모델을 마스터합니다.

* 7.1 [비지도 학습: 자율주행 텍스트 주제 분석과 올가미의 철학](/basic/07/sec01/)
  * 7.1.1 꼬리표 없는 야생 텍스트 방생 묶음과 분류 모형 세계관 충돌
  * 7.1.2 반반 치킨을 허용하지 않는 멍청한 강제 군집화 K-means의 물리적 맹점 파괴

* 7.2 [잠재축의 파괴적 차원 압축 해체: 잠재의미분석 (LSA)](/basic/07/sec02/)
  * 7.2.1 초거대 DTM 매트릭스 1,000만 배열을 3차원으로 썰어내는 특이값 분해(SVD)
  * 7.2.2 뉴스 업데이트 1개 유입에 전체 시스템 계산이 모조리 폭발하는 LSA 재부팅 저주

* 7.3 [패러다임의 역전 타임머신: 기계 조물주의 확률적 텍스트 생성](/basic/07/sec03/)
  * 7.3.1 무작위 주사위 조합으로 문서를 짜깁기하는 확률 공장 룰렛 시뮬레이션
  * 7.3.2 깡통 결과물을 손에 쥐고 과거의 시간 파라미터를 역으로 거슬러 올라가는 임무

* 7.4 [확률 역추적 톱니바퀴의 성배: 잠재 디리클레 할당 (LDA) 도입부](/basic/07/sec04/)
  * 7.4.1 반반 짬뽕 토픽 다중 혼합을 허락하는 기적의 문서 룰렛 지분표 ($\theta$ 세타)
  * 7.4.2 특정 토픽의 지독한 극단적 텍스트 스펠링 편식을 관장하는 고유 단어 주머니 ($\phi$ 파이)

* 7.5 [다면체 확률 자판기와 생성 건축 플레이트 표기법 (Plate Notation)](/basic/07/sec05/)
  * 7.5.1 삼각뿔 극단적 편식 분포와 섞어찌개 짬뽕을 맘대로 조율하는 알파($\alpha$) 세팅 마법
  * 7.5.2 컴퓨터의 루프 상상 도면의 노드 인과성을 구조화한 플레이트 아키텍처 노테이션 역산

* 7.6 [커닝 페이퍼의 미학: 깁스 샘플링(Gibbs) 지옥의 무한 MCMC 추론](/basic/07/sec06/)
  * 7.6.1 분석 의지 없이 무지성으로 처음 아무 데나 질러버리는 초기 난장판 스티커 할당
  * 7.6.2 내 방안 동거인 룸메이트 눈치 보기와 바깥 우주 쌍둥이의 도플갱어 눈치를 쓱 살피는 절대 최우도 커닝 조건부 잣대
