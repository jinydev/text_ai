# 7주차. 텍스트 주제 분석과 토픽 모델링

세상의 수만 장의 뉴스, 블로그 기사들이 어떤 공통된 카테고리를 은밀히 파벌을 이루며 공유하고 있는지를 기계 스스로 눈치채게 돕는 "비지도 학습"의 꽃! 잠재의미분석(LSA)의 행렬 분해 마법과, 조물주의 주사위를 역으로 훔쳐보는 디리클레 할당(LDA) 통계 역산 모델을 배웁니다.

* **[섹션 01. 비지도 학습: 자율주행 텍스트 주제 분석과 차원 집합의 철학](/basic/07/sec01/)**
  * 분류 모형과 군집 분석 모형의 철학적 전제 차이
  * 물리적 거리를 재는 K-means 클러스터링의 맹점
* **[섹션 02. 잠재축의 파괴적 차원 압축: 잠재의미분석 (LSA)](/basic/07/sec02/)**
  * DTM 매트릭스를 납작하게 써는 특이값 분해(SVD)
  * 새로운 데이터(오버피팅) 유입에 파괴되는 LSA의 연쇄 재부팅 버그
* **[섹션 03. 패러다임의 전환: 신의 확률적 문서 생성과 확률론의 역습](/basic/07/sec03/)**
  * 무작위 단어 조합으로 문서를 짜깁기하는 확률 공장 시뮬레이션
* **[섹션 04. 확률 역추적의 성배: 잠재 디리클레 할당 (LDA) 도입부](/basic/07/sec04/)**
  * 두 가지 수학적 파이프라인: 토픽 룰렛($\theta$)과 단어 셔플러($\phi$)의 분리 공정
* **[섹션 05. 디리클레 확률 판과 생성 플레이트 표기법 (Plate Notation)](/basic/07/sec05/)**
  * 극단적 편식과 박애 확장을 돕는 디리클레 분포 하이퍼파라미터
  * 컴퓨터의 확률 상상 과정을 도면화한 플레이트 설계 노테이션
* **[섹션 06. 커닝의 미학: MCMC 방식의 깁스 샘플링(Gibbs) 통계 역추적 추론](/basic/07/sec06/)**
  * 초기 난장판 스티커 할당과 무한 최적화 루프 검수기
  * 내 주변 동네 형제와 바깥 친척을 확률로 노려보는 2가지 조건부 방정식
