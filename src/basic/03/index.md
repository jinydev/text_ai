# 3주차. 텍스트 표현과 벡터 유사도

단어를 숫자로 바꾸어 공간에 흩뿌리고, 단어들 간의 수학적인 거리를 통해 유사도를 측정하는 고전 통계 방법론을 학습합니다.

* **[섹션 01. 단어 임베딩 기초와 고전 벡터 공간](/basic/03/sec01/)**
  * 컴퓨터는 문맹이다: 단어를 카운트 숫자로
  * 인간의 눈치 vs 고대 통계 모델의 벡터 맵핑
* **[섹션 02. 가장 단순한 치환: 원-핫 인코딩의 저주](/basic/03/sec02/)**
  * 단어의 완벽한 축 고립: 원-핫(One-Hot)
  * 희소성(Sparsity)과 차원 폭발의 우주적 메모리 낭비
* **[섹션 03. 문법 파괴자 Bag-of-Words(BoW)와 DTM 행렬](/basic/03/sec03/)**
  * 폭력성 100% 문법 파괴 가방 (BoW)
  * 다수 문서의 카운트 엑셀 표: DTM 빅데이터 표본
* **[섹션 04. 빈도 카운팅의 붕괴: 지프(Zipf)의 법칙](/basic/03/sec04/)**
  * 통계학적 빈부격차: 롱테일 마법과 The의 독식
  * 단순 카운팅 유사도의 완전한 논리적 패배
* **[섹션 05. 마법의 가중치 저울: TF-IDF 보정 방정식](/basic/03/sec05/)**
  * 흔한 쓰레기는 버리고 특수 키워드는 띄운다 (TF-IDF 원리)
  * 심화 수식: 자연로그(ln)와 스무딩(+1) 분모 방어막
* **[섹션 06. 벡터 공간 유사도 척도와 코사인(Cosine) 각도](/basic/03/sec06/)**
  * 똑같은 말을 반복하는 앵무새에게 속지 않는 코사인 방향
  * 피타고라스 유클리드와 자카드 집합 수리 비교
