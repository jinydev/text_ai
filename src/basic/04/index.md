# 4주차. 언어 모델 성능 평가

언어 모델이 얼마나 성능이 좋은지를 수학적으로 정량 평가하는 방법론들(PPL, BLEU, ROUGE 등)을 학습합니다.

* **[섹션 01. 언어모델: 고차원적 확률 맞추기 도박](/basic/04/sec01/)**
  * 컴퓨터는 확률 앵무새다: 자연어 채점의 본질
  * 번역기, 오타 교정, 음성 인식의 수학적 우위 전술
* **[섹션 02. 조건부 확률과 도미노 연쇄법칙(Chain Rule)](/basic/04/sec02/)**
  * 문장을 생성하는 근원: 고등학교 조건부 확률 공식
  * 도미노처럼 뻗어나가는 조건부 연속 곱셈($\prod$)
* **[섹션 03. 통계의 붕괴: 희소성의 폭발(Sparsity)](/basic/04/sec03/)**
  * 내 문장이 구글 서버 DB에 단 하나도 없다면?
  * 분모가 0이 되어 서버가 터져버리는 최악의 에러
* **[섹션 04. 근시안적 타협의 꼼수: N-gram 마르코프 체인](/basic/04/sec04/)**
  * 전체 문맥을 버리고 딱 N개만 보는 마르코프 타협
  * 치매 노인의 대화: 롱테일 의존성(명분) 상실 한계점
* **[섹션 05. 언어 모델 채점관: Perplexity(PPL) 분기 수학](/basic/04/sec05/)**
  * 객관식 vs 주관식 문제: 헷갈림 방어지수 찾기
  * 루트 분수식 철학 분해: 다음 단어 카드의 갯수
* **[섹션 06. 딥러닝 시대의 채점 4대장과 앵무새 환각](/basic/04/sec06/)**
  * PPL의 한계: 가장 그럴싸한 소설(환각, Hallucination)
  * 최신 지표 생태계: BLEU, ROUGE, MMLU, LLM-as-a-Judge
