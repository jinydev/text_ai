# 4. 언어 모델 (Language Model) 성능 평가와 수학적 통계의 붕괴

자연어 생성 모델이 얼마나 인간다운 문맥을 구사하는지 추론하는 수학적 정량 평가 방법론과, 과거 통계 모델들이 마주했던 거대한 연산의 장벽(Sparsity 에러)을 고찰합니다. 문장을 만들어내는 행위가 창조성이 아닌 고차원적 통계 도박(Branching Factor)임을 증명하고, 이를 수식화하는 연쇄 법칙 및 현대 LLM 시대의 필수 채점 체계(MMLU, BLEU, PPL)를 심층 학습합니다.

* 4.1 [언어 모델: 고차원적 확률 맞추기 통계 도박](/basic/04/sec01/)
  * 4.1.1 컴퓨터는 확률 앵무새다 (Stochastic Parrot)
  * 4.1.2 번역기, 자동 교정, 음성인식의 수학적 통계 전술

* 4.2 [조건부 확률과 도미노 연쇄 법칙 (Chain Rule)](/basic/04/sec02/)
  * 4.2.1 과거 SLM의 수학적 목표: 결합 확률 증명하기
  * 4.2.2 도미노처럼 뻗어나가는 무거운 연속 조건부 곱셈($\prod$)

* 4.3 [통계의 붕괴: 희소성의 치명적 폭발 (Sparsity)](/basic/04/sec03/)
  * 4.3.1 내 문장이 구글 서버 DB에 단 한 줄도 없다면?
  * 4.3.2 분모가 Zero가 되어 서버 전체가 먹통이 되는 최악의 에러

* 4.4 [근시안적 통계의 꼼수: N-gram과 마르코프 체인](/basic/04/sec04/)
  * 4.4.1 전체 문맥을 쓰레기통에 버리고 딱 N개만 보는 마르코프 단절
  * 4.4.2 롱테일 의존성(Long-Term Dependency) 상실 한계점 타파 고민

* 4.5 [수학적 헷갈림 채점관: Perplexity (PPL) 와 분기 분석](/basic/04/sec05/)
  * 4.5.1 객관식(비전) vs 주관식(자연어) 평가 구조의 괴리감 방어
  * 4.5.2 마르코프 역수 루트 분수식 철학: 기계가 쥔 다음 카드 방황 갯수 파악

* 4.6 [거대 딥러닝 시대의 채점 4대장과 환각 (Hallucination)](/basic/04/sec06/)
  * 4.6.1 단순 통계 PPL의 치명적 한계: 뻔뻔한 허언증(환각) 소설 생성
  * 4.6.2 거대 지식 생태계의 인증 마크: BLEU, ROUGE, MMLU, LLM-as-a-Judge
